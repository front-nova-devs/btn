# ✨ MotionType

A sleek and modern text animation component built with HTML, CSS, and JavaScript. Featuring smooth blur-in transitions and a premium UI aesthetic.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🚀 Overview

**MotionType** is a lightweight, easy-to-use component for creating elegant text reveal animations. It breaks down text into individual characters and animates them with a sophisticated blur and scale effect, perfect for landing pages, headers, or hero sections.

## ✨ Features

- **Character-by-character animation**: Each letter animates in sequentially with a smooth blur + scale effect.
- **Modern Design**: Minimalist and premium look with subtle radial gradients.
- **Fully Customizable**: Easily change text, colors, and animation speeds via CSS variables and JavaScript.
- **Responsive & Lightweight**: Pure HTML/CSS/JS with no heavy dependencies.
- **Interactive**: Includes a refresh button to replay the animation on demand.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Modern layout, transitions, and `@keyframes` animations.
- **JavaScript**: Dynamic character manipulation and interaction logic.
- **Google Fonts**: Utilizing *Google Sans Flex* and *Plus Jakarta Sans* for premium typography.

## ⚙️ Customization

### Change the Text
In `index.html`, modify the `text` constant:
```javascript
const text = "Welcome to MotionType";
```

### Change Styles
Update the CSS variables in `style.css` to match your brand:
```css
:root {
  --bg-color: #0f172a;
  --primary: #3b82f6;
  --text-primary: #f8fafc;
}
```

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/front-nova-devs/motion-type.git
   ```
2. Open `index.html` in your favorite browser.
